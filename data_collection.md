# Data Collection & Integration

## 1. Data Sources
The primary data source for this project is the **Synthetic Mobile Money Transactions for Fraud Detection Research** dataset from Kaggle.
* **URL:** [Financial Fraud Detection Dataset](https://www.kaggle.com/datasets/sriharshaeedala/financial-fraud-detection-dataset)
